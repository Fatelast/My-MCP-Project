# UI/UX 设计规范文档

## 摘要

- **设计风格**：深色主题牌桌风格，参考实体麻将桌的绿色绒面质感，现代简洁
- **主色调**：深绿 `#1B5E20`（牌桌）/ 象牙白 `#FFF8E7`（牌面）/ 金色 `#FFD700`（强调）
- **核心组件**：Tile（麻将牌）、PlayerArea（玩家区域）、ActionPanel（操作面板）、ScoreModal（结算弹窗）
- **响应式断点**：Mobile < 640px / Tablet 640-1024px / Desktop > 1024px
- **设计系统**：Tailwind CSS + 自定义 CSS 变量，不依赖 UI 组件库

---

## 1. 设计理念

> 让玩家专注于游戏本身——牌桌干净、牌面清晰、操作直觉。

### 设计目标

| 目标 | 描述 | 衡量标准 |
|------|------|----------|
| 易用性 | 玩家无需说明书即可开始游戏 | 新手 30 秒内找到出牌方式 |
| 清晰性 | 任何尺寸屏幕上牌面清晰可辨 | 移动端牌面 ≥ 24px 宽 |
| 沉浸感 | 牌桌视觉有真实感 | 深色绿色绒面质感 |

## 2. 设计系统

### 2.1 颜色系统

```css
:root {
  /* 牌桌 */
  --color-table:        #1B4332;  /* 深绿，主牌桌背景 */
  --color-table-light:  #2D6A4F;  /* 亮绿，区域边界 */

  /* 牌面 */
  --color-tile-bg:      #FFF8E7;  /* 象牙白，牌面背景 */
  --color-tile-border:  #D4B896;  /* 米色，牌边框 */
  --color-tile-shadow:  rgba(0,0,0,0.4);

  /* 字色 */
  --color-man:          #C62828;  /* 万子：红色 */
  --color-pin:          #1565C0;  /* 饼子：蓝色 */
  --color-sou:          #2E7D32;  /* 索子：绿色 */
  --color-honor:        #4A148C;  /* 字牌：紫色 */
  --color-aka:          #FF1744;  /* 赤牌：亮红 */

  /* UI */
  --color-primary:      #FFD700;  /* 金色，主要按钮/强调 */
  --color-primary-dark: #F9A825;
  --color-danger:       #EF5350;  /* 危险操作 */
  --color-text-main:    #F5F5F5;  /* 主文字 */
  --color-text-muted:   #9E9E9E;  /* 次要文字 */
  --color-overlay:      rgba(0,0,0,0.7);
}
```

### 2.2 字体系统

```css
--font-game: 'Noto Sans JP', 'PingFang SC', sans-serif; /* 支持日文汉字 */
--font-mono: 'SF Mono', monospace;  /* 点数显示 */
```

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 牌面大 | 20-28px | 700 | 桌面端牌面文字 |
| 牌面小 | 14-18px | 700 | 移动端牌面文字 |
| 分数 | 18px | 600 | 玩家分数 |
| 标签 | 12px | 400 | 场风/自风/番数 |

### 2.3 间距系统

基础单位 4px，使用 Tailwind 间距类。

### 2.4 动效系统

| 场景 | 动效 | 参数 |
|------|------|------|
| 选中手牌 | 向上位移 | translateY(-8px), 150ms ease |
| 出牌动画 | 滑入河牌区 | 200ms ease-out |
| 摸牌动画 | 从牌山滑入 | 150ms ease |
| 操作按钮出现 | 弹入 | scale 0.8→1, 200ms ease-out |
| 和牌结算弹窗 | 淡入+缩放 | scale 0.9→1, 300ms ease-out |
| AI思考 | 脉冲动画 | 1s infinite |

## 4. 核心组件规范

### 4.1 麻将牌（Tile 组件）

#### 尺寸规格

| 场景 | 宽 | 高 | 字号 |
|------|----|----|------|
| 玩家手牌（桌面） | 36px | 48px | 22px |
| 玩家手牌（手机） | 28px | 38px | 16px |
| 对手手牌（背面） | 24px | 32px | - |
| 副露/河牌 | 28px | 38px | 16px |
| 宝牌指示牌 | 24px | 32px | 14px |

#### 视觉规范

```
┌──────┐
│  ①   │  ← 牌面文字（Unicode 或数字+花色符号）
│   饼  │  ← 花色标注（可选）
└──────┘
```

- 背景：象牙白 `#FFF8E7`，轻微渐变（顶部稍白）
- 边框：1px solid `#D4B896`，圆角 3px
- 阴影：`2px 3px 4px rgba(0,0,0,0.35)`（产生立体感）
- 选中状态：向上位移 8px + 金色边框 `2px solid #FFD700`
- 悬停状态：向上位移 4px，transition 150ms

#### 牌面内容（Unicode 渲染）

| 花色 | 显示内容 | 颜色 |
|------|----------|------|
| 万子 1-9 | １万 ～ ９万 | `#C62828` 红色 |
| 饼子 1-9 | ①～⑨ | `#1565C0` 蓝色 |
| 索子 1-9 | 1竹 ～ 9竹 | `#2E7D32` 绿色 |
| 东南西北 | 東南西北 | `#4A148C` 紫色 |
| 中发白 | 中/發/白 | 中=红/發=绿/白=灰 |

### 4.2 操作面板（ActionPanel）

#### 布局

```
┌────────────────────────────────────────┐
│  [跳过]  [吃 ▼]  [碰]  [杠]  [荣/自摸]  │
└────────────────────────────────────────┘
```

- 固定在玩家手牌区下方
- 按钮出现时有弹入动画
- 5秒倒计时进度条（超时自动跳过）

#### 按钮规格

| 按钮 | 颜色 | 优先级 | 尺寸 |
|------|------|--------|------|
| 荣/自摸 | 金色 `#FFD700` | 最高，放最右 | 64×40px |
| 碰/杠 | 橙色 `#FF8F00` | 高 | 56×40px |
| 吃 | 蓝色 `#1565C0` | 中 | 56×40px |
| 跳过 | 灰色 `#616161` | 低，放最左 | 56×40px |

## 5. 响应式设计规范

| 断点 | 布局策略 |
|------|----------|
| > 1024px | 四方向完整牌桌，中央信息区 |
| 640-1024px | 压缩牌面尺寸，保持四方向布局 |
| < 640px | 上/左/右家缩略显示，玩家手牌横向滚动，优先操作区 |

**移动端特殊处理**：
- 玩家手牌区支持横向滑动
- 操作按钮固定在屏幕底部
- 建议横屏游玩（显示横屏提示）
- 触控目标最小 44×44px

## 7. 无障碍规范

- 颜色对比度 ≥ 4.5:1（牌面文字/背景）
- 所有操作按钮有 `aria-label`
- 键盘可操作（Tab 选牌，Enter 出牌）
- 触控目标 ≥ 44×44px
